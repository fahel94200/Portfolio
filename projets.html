Projet BTS SIO

Sicily Lines – Réservation de Traversées Maritimes

Durée: 2 mois

C# Windows Forms PHP Laravel SQL MySQL UML Merise
Description

Conception et développement complet d'un système de réservation de traversées maritimes pour la compagnie Sicily Lines. Le projet comprend la modélisation UML, la conception de la base de données relationnelle, ainsi que le développement de deux applications de réservation : une application desktop en C# avec Windows Forms et un site web en PHP Laravel.

Fonctionnalités principales
  • Modélisation UML avec diagramme de classes complet
  • Base de données relationnelle MySQL avec 9 tables interconnectées
  • Application desktop en C# Windows Forms pour la gestion des réservations
  • Site web de réservation en PHP Laravel (architecture MVC)
  • Gestion des secteurs géographiques et liaisons maritimes
  • Gestion de la flotte de bateaux et leurs capacités
  • Système de traversées programmées avec disponibilités
  • Tarification dynamique par période (basse, moyenne, haute saison)
  • Gestion des réservations clients avec lignes de détail
Compétences acquises
  • Développement d'applications desktop en C# (Windows Forms)
  • Développement web full-stack avec PHP Laravel
  • Modélisation UML (diagrammes de classes)
  • Conception de bases de données relationnelles
  • Langage SQL avancé (DDL, DML, contraintes)
  • Analyse et modélisation des besoins métier
